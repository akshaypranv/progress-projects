# 📁 Mondrian Project: Asymmetric Grid Architecture 🎨

This project is a technical study in **Abstract UI Composition** using the **CSS Grid** engine. It replicates the iconic asymmetric "Mondrian" style to demonstrate advanced control over grid tracks, spanning logic, and negative space management.

### 🛠️ Technical Concepts Used

* **Explicit Grid Definition:** Utilizing `grid-template-columns` and `grid-template-rows` to establish a non-uniform coordinate system.
* **Line-Based Placement:** Implementation of `grid-column` and `grid-row` shorthand (e.g., `span 2 / 3`) to allow elements to occupy multiple tracks and overlap logically.
* **Fractional Geometry:** Leveraging `fr` units alongside fixed pixel values to maintain structural integrity across different viewport scales.
* **Box Model Precision:** Exacting control over `border` widths and `gap` properties to achieve the "heavy-stroke" aesthetic characteristic of De Stijl art.

### 🎯 Objective

To move beyond standard "symmetrical" grids and master the ability to build complex, irregular layouts that require precise coordinate-based element placement.
