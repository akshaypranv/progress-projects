# 💳 Responsive Pricing Architecture: Flexbox Implementation

A highly functional, responsive UI component designed to demonstrate **Fluid Layout Transitions**. This project replicates a real-world SaaS pricing table, focusing on user experience across mobile and desktop environments.

### 🛠️ Technical Logic
- **Dynamic Flexbox Containers:** Implemented `display: flex` with a mobile-first mindset. 
- **Fluid Layout Switching:** Utilized `flex-direction: column` for vertical stacking on mobile and `flex-direction: row` for horizontal expansion on desktop.
- **Breakpoints:** Integrated CSS Media Queries to trigger layout shifts precisely when the viewport expands, ensuring a seamless user transition.
- **Visual Hierarchy & Styling:** Applied custom card styling including shadows, hover effects, and typography scaling to distinguish between pricing tiers.

### 🧩 Key Concepts Mastered
- **Flex-Wrap & Direction:** Controlling the flow of content based on screen real-estate.
- **Spacing & Alignment:** Using `gap` and `justify-content` to maintain professional white space.
- **Component-Based Styling:** Designing reusable "Pricing Cards" with a clean, modern aesthetic.

### 🎯 Project Objective
To master the **"Contract and Expand"** logic of modern web development. This proves my ability to build components that are "production-ready" for both smartphone and wide-screen desktop users.
