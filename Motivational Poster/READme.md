# Motivational Poster: Visual Hierarchy 🖼️

A design-to-code project focused on creating a clean, high-contrast UI. This project emphasizes the balance between imagery and typography to convey a message.

### 🛠️ Technical Concepts Used
- **Typography:** Integration of custom font families and hierarchy (h1, p).
- **Styling:** Customizing text shadows, borders, and margins to replicate a physical poster feel.
- **Asset Integration:** Efficiently rendering high-quality images as the focal point of the UI.

### 🎯 Objective
To practice aesthetic styling and ensuring that the CSS effectively communicates the "vibe" of the content through spacing and color contrast.
