# Capstone 1: Multi-Page Web Architecture 🏛️

A comprehensive project focused on building a multi-page site using only semantic HTML5. This project serves as a blueprint for understanding how a browser links and parses various document trees.

### 🛠️ Features
- **Multi-page Navigation:** Integrated internal linking between `index.html` and `contact.html`.
- **Semantic Structure:** Utilization of `<header>`, `<main>`, `<footer>`, and nested section tags.
- **Media Management:** Organized directory structure for local images.
- **Form Handling:** Basic contact form structure and input types.

### 🎯 Objective
To master the fundamentals of a "Complete Website" before moving into Single Page Applications (SPAs) like React.
